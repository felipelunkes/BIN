Sistema Operacional Bin S.O

O Bin S.O é um sistema operacional 16-Bits desenvolvido para processadores 8086 ou superiores. O Bin S.O opera em modo real e aceita o desenvolvimento de aplicativos em linguagem assembly utilizando a IDE do Bin S.O.

Desenvolvendo aplicativos:

Uma IDE foi especialmente desenvolvida para possibilitar o desenvolvimento de aplicativos em assembly para o Bin S.O. A IDE oferece todo o ambiente necessário para que a criação de aplicativos seja possível. A IDE inclui o compilador assembly, o montador de templates e outros arquivos e aplicativos úteis para a instalação e manutenção. Acesse https://github.com/felipelunkes/binIDE para obter a IDE oficial do Bin S.O. Após obtê-la, descompacte-a e execute ./configure para instalar o restante dos pacotes necessários.

Requisítos mínimos:

> Processador Intel 8086 ou superiores ( ou AMD x86 ).

> 1 MB de memória RAM.

> Teclado padrão.

> Monitor.

> Disco Rígido ou Pen Drive de 1 MB.

Instalando o Bin S.O em uma mídia externa:

Utilize para instalar o Bin S.O em um computador ou mídia de armazenamento externa como root:

./instalar dispositivo binloader binkernel ajuda sobre ver mem processador clear video echo reg desligar reiniciar som binmouse
