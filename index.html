*** Sistema Operacional Bin S.O

O Bin S.O é um sistema operacional 16 Bits desenvolvido para processadores Intel 8086 ou superiores, incluindo processadores AMD que utilizem a plataforma x86. O Bin S.O opera em modo real e aceita o desenvolvimento de aplicativos em linguagem assembly utilizando a IDE do Bin S.O. O Bin S.O apresenta suporte a alguns dispositivos PCI, caixa de som e envio de dados via portas seriais.

O suporte a caixa de som do computador permite que o sistema emita sons de alerta e/ou avisos. O suporte a dispositivos PCI permite que o sistema se comunique com um número limitado de dispositivos. O suporte a portas seriais e paralelas permite que o sistema possa receber e enviar informações e até mesmo imprimir mensagens por impressoras conectadas a estas portas.

Desenvolvendo aplicativos:

Uma IDE foi especialmente desenvolvida para possibilitar o desenvolvimento de aplicativos em assembly para o Bin S.O. A IDE oferece todo o ambiente necessário para que a criação de aplicativos seja possível. A IDE inclui o compilador assembly, o montador de templates e outros arquivos e aplicativos úteis para a instalação e manutenção. Acesse http://felipelunkes.github.com/binIDE/ para obter a IDE oficial do Bin S.O. Após obtê-la, descompacte-a e execute a++ para iniciar a IDE. A IDE oferece um sistema de atualizações para que o desenvolvedor obtenha sempre as bibliotecas mais recentes para o desenvolvimento eficiente e fácil. Os pacotes e aplicativos inclusos da IDE são atualizados juntamente com a IDE. A IDE oferecida foi desenvolvida para a arquitetura i386. Assim, pode ser executada em computadores baseados em i386 e x86-x64.

A IDE do Bin S.O está disponível para sistemas operacionais baseadas no Windows, na versão 1.0 Alpha e no CD-ROM de instalação do Bin S.O, sobre Linux, na versão 0.1 Alpha.

Vantagens:

Memória RAM:

O Bin S.O utiliza apenas 1 MB de Memória RAM para seu funcionamento e de aplicativos do usuário, que são executados em tempo real. O sistema apresenta uma grande velocidade de execução.

Processador:

O Bin S.O pode ser utilizado em processadores modernos, mas também pode ser executado em processadores e sistemas mais modestos. Ele foi desenvolvido para ser rápido em processadores mais antigos e adaptável a processadores atuais que suportem sua arquitetura.

Requisítos mínimos:

> Processador Intel 8086 ou superiores ( ou AMD x86 ).

> 1 MB de memória RAM ( Mínimo ).

2 MB de memória RAM ( Recomendado ).

> Teclado padrão.

> Monitor 16 Cores.

> Disco Rígido ou Pen Drive de no mínimo 5 MB.

10 MB de armazenamento recomendados para Disco Rígido ou qualquer mídia de armazenamento.

Aparência do Bin S.O:

O Bin S.O possui uma interface denominada CLI ( Command Line Interface ) ou simplesmente Interface de Linha de Comando.

Instalando o Bin S.O em uma mídia externa:

Utilize para instalar o Bin S.O em um computador ou mídia de armazenamento externa:

Você deve possuir o disco de instalação do Bin S.O, que pode ser encontrado em https://felipelunkes.github.com/Imagens . Este disco de instalação do Bin S.O possui em sistema Linux mínimo responsável por instalar o Bin S.O corretamente em um disco de armazenamento válido. Ele também possui uma versão da IDE específica e as ferramentas de compilação e instalação de novos aplicativos. Este sistema Linux foi desenvolvido especificamente para servir de instalador para o Bin S.O. Ele foi desenvolvido por Felipe Miguel Nery Lunkes. Após realizar o download, use-o para instalar o Bin S.O. Ao iniciar pelo CD, as instruções serão exibidas na tela.
